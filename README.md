# CTF Write-Ups Portfolio

This repository contains curated Capture The Flag (CTF) write-ups demonstrating
practical skills in vulnerability analysis, exploitation, and secure coding concepts.

The goal of these write-ups is to clearly explain **what the vulnerability was,
why it existed, and how it was exploited**, with an emphasis on defensive lessons
and mitigation.
